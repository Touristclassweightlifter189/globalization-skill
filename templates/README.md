# Templates

Templates define the minimum structure for recurring globalization artifacts. Start with the template, then compare it against the example section inside the same file.

Recommended order:

1. `locale-resolution-policy.template.md`
2. `capability-matrix.template.md`
3. `privacy-governance-matrix.template.md`
4. `market-intake.template.md`

Use templates for product-specific decisions. Keep universal rules in `references/`, not in filled templates.
